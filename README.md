# Fitness Page

Fitness Page is a website that provides users with information on exercises, and overall fitness. The website is built using HTML, CSS, and JavaScript.

## 1. Introduction

Fitness Page is a website designed to help users improve their overall fitness. The website provides information on exercises, and other fitness-related topics. 

The website is built using `HTML`, `CSS`, and `JavaScript`. It is fully responsive and works on desktop and mobile devices.

## 2. Features

- Information on exercises, and other fitness-related topics.
- Fully responsive design that works on desktop and mobile devices.


## 3. Installation

To run Fitness Page on your local machine, you can clone the repository:
- `git clone https://github.com/Artur-Vanivskyi/Fitness-page.git`
- `cd Fitness`
- `npm install`
- `npm start`

## 4. Usage: Accessing Exercise Videos

Fitness Page provides links to exercise videos on YouTube and other websites. To access the videos, simply follow these steps:

1. Navigate to the Exercises page by clicking on the "Exercises" link in the navigation bar at the top of the page.

2. Scroll down to the list of exercises. Each exercise is accompanied by a thumbnail image and a short description.

3. Click on the exercise that you are interested in. This will take you to a page with more information about the exercise.

4. On the exercise page, you will see a video section. This section contains a link to a video that demonstrates how to perform the exercise.

5. Click on the link to watch the video. The video will open in a new tab or window, depending on your web browser.
